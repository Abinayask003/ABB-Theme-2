# NL2PLC Prototype

This project converts natural-language PLC instructions into IEC 61131-3 Structured Text (.st) code using **Ollama (local LLM)**.

## Features
- Enter single or batch instructions
- Generate IEC 61131-3 ST code
- Save `.st` files
- Local simulation
